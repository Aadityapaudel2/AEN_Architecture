# Reproducibility Notes

This staging folder is pinned to the Zenodo release
`v1.0-preprint-2026-04-22`.

The `paper/` directory is extracted directly from the Zenodo source zip. Later
local wording edits are intentionally excluded.

Runtime datasets, raw transcripts, private notebooks, and local experiment
stores are not bundled here. The paper references them as evidence artifacts,
but this GitHub stage is limited to the Zenodo paper source package and the
Zenodo PDF.
