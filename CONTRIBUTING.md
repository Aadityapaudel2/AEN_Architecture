# Contributing

Use branches and pull requests for edits. Keep empirical claims tied to local artifacts, diagnostics, or clearly marked plans.

For paper edits:

1. Edit files under `paper/`.
2. Rebuild with `./build.ps1`.
3. Include the updated rendered PDF in `artifacts/AEN_preprint.pdf` when making a release candidate.
4. Note whether changes affect claims, figures, citations, or only wording.

Do not add external citations unless the source has been inspected and accepted for this paper.
