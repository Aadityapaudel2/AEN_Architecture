## Suggested Description

This preprint introduces Artificial Evaluation Network (AEN), a runtime architecture for exact-answer mathematical reasoning with fixed-weight language models. AEN combines Runtime-at-Boot certification, role-specific boot memory, long-context Qwen-family serving, a triadic Athena-Aria-Artemis controller protocol, controller-owned finalization, Canon v2.1 YAML-style distillation, and artifact-based evaluation. The paper reports Vault of Echoes diagnostics, AIME 2026 frozen-notebook and unrestricted one-loop runs, and a scaling plan for higher-compute AEN evaluations.
